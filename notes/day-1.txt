Binding:
	data: helps you bind component data in the template. the flow of data is from code to the UI
	syntax: 
	{{prop-name}} -> interpolation [stringifies the property value]
	<element [attribute-name]="prop-name"> -> property binding [does not stringify]
	event: helps you to bind an event of an element with a method in the component code, through which data can be passed from UI to code
	syntax: <element (event-name)="method-of-code(args)">
